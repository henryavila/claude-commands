## Knowledge Base de Técnicas de Skills

Análise profunda do plugin superpowers v5.0.5 (14 skills, 36 arquivos).
Documentação completa em `docs/kb/`:

- **`analise-superpowers-v5.0.5.md`** — Análise completa: 23 técnicas catalogadas (T01–T23), taxonomia de skills, padrões estruturais, anti-padrões, mapa de integração, notas para rastreamento de atualizações
- **`templates-reutilizaveis.md`** — 17 templates (TPL-01–TPL-17) prontos para copiar e adaptar: Iron Law, HARD-GATE, prompts de subagentes, cenários de pressão, flowcharts, etc.

**Quando consultar:**
- Ao criar ou melhorar `hca-` commands
- Ao avaliar atualizações do superpowers ou outros plugins de skills
- Ao precisar de um template de prompt para subagentes, review loops ou regras disciplinadoras
